# The Collector Productivity Call Flow Toolkit Landing Page

Static public landing page for **The Collector Productivity Call Flow Toolkit**, a digital debt collector training product.

## Repository contents

This repository is intended to contain public landing page files only:

- `index.html`
- `styles.css`
- `README.md`

Do not add private product files, the actual PDF/toolkit, passwords, API keys, customer data, Payhip account details, bank information, or private notes.

## Checkout link placeholders

Replace these placeholder links before launch:

- `INDIVIDUAL_PAYHIP_LINK`
- `TEAM_PAYHIP_LINK`
- `AGENCY_INQUIRY_LINK`

They appear in `index.html` and are intentionally easy to search and replace.

## Local preview

Open `index.html` in a browser to preview the page. No build step or paid dependency is required.

## Enable GitHub Pages from the main branch root

1. Merge the landing page pull request into `main`.
2. Open the repository on GitHub.
3. Select **Settings**.
4. In the left sidebar, select **Pages**.
5. Under **Build and deployment**, set **Source** to **Deploy from a branch**.
6. Set **Branch** to `main`.
7. Set the folder to `/ (root)`.
8. Select **Save**.
9. Wait for GitHub Pages to publish the site.
10. Use the Pages URL shown in the GitHub Pages settings panel.

## Launch checklist

- Replace all three placeholder checkout links.
- Update the canonical URL and Open Graph URL/image in `index.html`.
- Confirm the page does not include private product files or sensitive information.
- Review compliance language with qualified counsel or internal compliance leadership before public launch.
